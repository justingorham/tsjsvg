<!-- usage -->
```sh-session
$ npm install -g @justingorham/ts-json-schema-validator-generator
$ tsjsvg COMMAND

$ tsjsvg (-v|--version|version)
@justingorham/ts-json-schema-validator-generator/1.0.0 win32-x64 node-v12.14.1
$ tsjsvg --help [COMMAND]
USAGE
  $ tsjsvg COMMAND
...
```
